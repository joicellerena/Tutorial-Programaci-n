# Tutorial de Programación

Este repositorio contiene un tutorial básico publicado con GitHub Pages (carpeta docs/).

URL propuesta para Pages: https://joicellerena.github.io/Tutorial-Programaci-n/

Estructura:
- docs/ : páginas del tutorial
- exercises/python/ : ejercicios y soluciones en Python

Nota: Si GitHub Pages no aparece activo, ve a Settings → Pages y selecciona la rama main y la carpeta /docs como fuente.
